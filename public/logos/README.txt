LOGOS DE PROVEEDORES PARA PREFACTURAS
=====================================

Coloca aquí los logos de cada proveedor para que aparezcan en el PDF
de la prefactura (esquina superior derecha, donde dice el nombre).

CÓMO NOMBRAR LOS ARCHIVOS
-------------------------
Convierte el nombre del proveedor a minúsculas, sin acentos y con
guiones en lugar de espacios. Extensión recomendada: .png o .jpg

Ejemplos:
  ADET                  →  adet.png
  FAST INTEGRAL         →  fast-integral.png
  KEKO                  →  keko.png
  PAQUETELIT            →  paquetelit.png
  VERDE DISEÑO LOGISTIC →  verde-diseno-logistic.png
  STAR KG               →  star-kg.png
  PARTRUNNER            →  partrunner.png
  TIL                   →  til.png
  RAPI2                 →  rapi2.png

RECOMENDACIONES DE IMAGEN
-------------------------
- Tamaño ideal: 300x150 px (proporción 2:1)
- Fondo transparente (PNG) o blanco (JPG)
- Resolución suficiente para impresión: 150 DPI

EXTENSIONES SOPORTADAS (en orden de búsqueda)
---------------------------------------------
1. .png
2. .jpg
3. .jpeg
4. .svg
5. .webp

Si el logo no se encuentra, el PDF muestra el nombre del proveedor en
texto como fallback (sin error).
